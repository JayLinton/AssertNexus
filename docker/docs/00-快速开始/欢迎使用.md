# 欢迎使用 Assert Nexus

这是一个自托管的知识库文档站，帮助你快速搭建个人或团队的文档系统。

## 特性

- 📝 Markdown 文档支持
- 🔍 全文搜索（支持拼音）
- 🤖 AI 助手（可选，需配置 API Key）
- 📱 移动端适配
- 🎨 简洁美观的界面
- 🚀 一键 Docker 部署

## 快速开始

1. 编辑 `.env` 文件，配置你的站点信息
2. 将你的 Markdown 文档放入 `docs/` 目录
3. 运行 `docker-compose up -d`
4. 访问 `http://localhost:3000`

## 文档目录结构

```
docs/
├── 00-快速开始/
│   ├── 欢迎使用.md      ← 你正在看的文档
│   └── 如何添加文档.md
├── 01-Markdown示例/
│   ├── 基础语法.md
│   └── 代码块与高亮.md
└── 02-进阶功能/
    ├── AI助手配置.md
    └── 搜索功能.md
```

文件夹名称前的数字用于排序，格式为 `XX-名称`。
