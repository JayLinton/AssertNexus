# 如何添加文档

## 添加新文档

1. 在 `docs/` 目录下创建或编辑 `.md` 文件
2. 文件会自动被识别，无需重启服务
3. 刷新浏览器即可看到更新

## 文件夹命名规则

```
docs/
├── 00-文件夹名/     ← 数字前缀用于排序
├── 01-另一个文件夹/
└── 02-第三个文件夹/
```

- 文件夹名称格式：`XX-名称`（XX 为两位数字）
- 数字越小，排序越靠前
- 不带数字前缀的文件夹会排在最后

## 文档命名规则

- 使用 `.md` 扩展名
- 文件名即为文档标题
- 支持中文文件名

示例：
```
docs/
└── 01-Python基础/
    ├── 变量与数据类型.md
    ├── 流程控制.md
    └── 函数定义.md
```

## 文档内容

每个文档的第一行 `# 标题` 会作为文档标题显示在页面上。

```markdown
# 这是文档标题

这是文档内容...
```

## 实时更新

- 添加/修改文档后，刷新浏览器即可看到变化
- 无需重启 Docker 容器
- 搜索索引会自动更新（有 5 分钟缓存）
