# 搜索功能

## 快速搜索

### 快捷键

- `Ctrl + K` 或 `⌘ + K`：打开搜索框
- `ESC`：关闭搜索

### 搜索方式

1. **关键词搜索**：输入文档中的关键词
2. **拼音搜索**：输入中文的拼音全拼
3. **首字母搜索**：输入拼音首字母

### 示例

假设有一篇文档《Python 基础语法》：

| 输入 | 结果 |
|------|------|
| Python | ✓ 匹配 |
| python | ✓ 匹配（不区分大小写） |
| py | ✓ 匹配（首字母） |
| Python基础 | ✓ 匹配 |
| python jichu | ✓ 匹配（拼音） |
| pyjc | ✓ 匹配（拼音首字母） |

## 搜索结果

- 显示文档标题和所属文件夹
- 高亮匹配的关键词
- 显示内容摘要
- 点击直接跳转到文档

## 搜索范围

搜索会覆盖：
- 文档标题
- 文档内容
- 文件夹名称

## 技术说明

- 搜索 API 基于文件系统实时扫描
- 结果有 5 分钟缓存
- 支持中英文混合搜索
- 拼音搜索基于 pinyin-pro 库
